About

Random Forest Classifier

Random Forest Classifier builds multiple decision trees where each node of the tree asks a certain question or uses a certain criteria to decide which path the input value travels to and finally reaches the end node. The question and where it is asked is decided by the Gini index criterion. Once, multiple trees have been built, The algorithm uses the voting system to predict the final answer

Naive Bayes

The Naive Bayes Algorithm works on the principle of Bayes theorem form probability. It Calculates the probability of Y given X based on the probability of getting a Y class and Probability of getting the given X given the Y class.

Ensemble Stacking

We have used 2 different models for this task. each classifier may have it's own disadvantages. Stacking is basically using both the models to predict the probability the user like a certain movie and then using either averaging mechanism or voting mechanism to complement for other's disadvantages.

Possible disadvantages of this type of modelling

  • The Meaning behind the sequence of the data might be missed
  • The training dataset is relatively smaller compared to many real life datasets and the words that are not in the dataset are not considered while performing classification
